Personal Budget & Expense Tracker
Project Overview

This project is the first stage of a Personal Budget & Expense Tracker. It provides the basic HTML structure and CSS styling that will be expanded in future weeks into a fully functional web application.

Features
A main heading and short project description.
An Add Expense section with input fields for:
Expense Name
Amount
Category
A Your Expenses section with placeholder text ("No expenses yet").
Basic styling using element, class, and ID CSS selectors.
HTML and CSS organized in separate files.
Technologies Used
HTML5
CSS3
Project Structure
Personal-Budget-Tracker/
│
├── index.html
├── style.css
└── README.md
Learning Objectives

This assignment demonstrates:

Creating a well-structured HTML page.
Linking an external CSS stylesheet.
Using element, class, and ID selectors.
Organizing a simple web project.
Exploring and testing styles using browser DevTools.
Future Improvements

In the coming weeks, this project will be enhanced by:

Adding JavaScript functionality.
Allowing users to add and remove expenses.
Displaying a list of expenses dynamically.
Calculating totals and remaining budget.
Saving data for future use.
